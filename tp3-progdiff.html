Nous explorerons galement lutilisation de LRScheduler et de LRSchedulerOnPlateau pour ajuster dynamiquement le taux dapprentissage afin doptimiser la convergence du modle.

My Tensor

Dans cette section, nous dfinissons une classe Tensor, qui reprsente une variable tensorielle de base avec sa valeur (donne) et son gradient. Cette classe est conue pour tre utilise dans des calculs diffrentiables, permettant ainsi de suivre les gradients pendant l'entranement de modles paramtriques, comme les rseaux de neurones.

La classe Tensor contient plusieurs mthodes qui permettent d'effectuer des oprations sur des objets Tensor, comme l'addition, la multiplication, la puissance, et l'activation ReLU. Chaque opration est accompagne d'un calcul de gradient via la mthode backward.

Attributs

  • data : La valeur du tensor (un scalaire).
  • grad : Le gradient du tensor, initialis 0.
  • _prev : Un ensemble qui stocke les tensors prcdents ncessaires pour la rtropropagation.
  • _op : L'opration ayant gnr ce tensor (par exemple +, *, etc.).

Mthodes

  1. Addition (__add__) : Effectue l'addition de deux tensors.
  2. Multiplication (__mul__) : Effectue la multiplication de deux tensors.
  3. Puissance (__pow__) : Permet d'lever un tensor une puissance spcifie.
  4. ReLU (relu) : Applique la fonction d'activation ReLU, qui met zro les valeurs ngatives.
  5. backward() : Calcule les gradients de tous les tensors dans le graphe de computation en utilisant la mthode de rtropropagation.

class Tensor:
    
    """ stores a single scalar Tensor and its gradient """

    def __init__(self, data, _children=(), _op=''):

        self.data = data
        self.grad = 0.0

        self._backward = lambda: None
        self._prev = set(_children)
        self._op = _op

    def __add__(self, other):
        other = other if isinstance(other, Tensor) else Tensor(other)

        out = Tensor(self.data + other.data, (self, other), '+')

        def _backward():
            self.grad += out.grad
            other.grad += out.grad

        out._backward = _backward

        out._prev = set([self, other])
        return out

    def __mul__(self, other):
        
        other = other if isinstance(other, Tensor) else Tensor(other)

        out = Tensor(self.data * other.data, [self, other], '*')

        def _backward():
            self.grad += other.data * out.grad
            if other.data != 0:
                other.grad += self.data * out.grad
            else:
                other.grad = 0
        out._backward = _backward

        return out

    def __pow__(self, other):

        assert isinstance(other, (int, float)), "only supporting int/float powers for now"

        out = Tensor(self.data**other, (self,), f'**{other}')

        def _backward():
            self.grad += (other * self.data**(other-1)) * out.grad
            
        out._backward = _backward

        return out

    def relu(self):
        out = Tensor(0 if self.data < 0 else self.data, [self], 'ReLU')
        def _backward():
            self.grad += out.grad
        out._backward = _backward
        return out
    
    def build_topo(self, visited=None, topo=None):
        if self not in visited:
            visited.add(self)
            for child in self._prev:
                child.build_topo(visited=visited, topo=topo)
            topo.append(self)
        return topo

    def backward(self):
        topo = []
        visited = set()
        topo = self.build_topo(topo=topo, visited=visited)
        
        self.grad = 1.0
        for v in reversed(topo):
            v._backward()

    def __neg__(self): # -self
        return self * -1

    def __radd__(self, other):
        return self + other

    def __sub__(self, other):
        return self + (-other)

    def __rsub__(self, other):
        return other + (-self)

    def __rmul__(self, other):
        return self * other

    def __truediv__(self, other):
        return self * other**-1

    def __rtruediv__(self, other):
        return other * self**-1

    def __repr__(self):
        return f"Tensor(data={self.data}, grad={self.grad})"

4.1 Stochastic Gradient Descent (SGD)

La mthode SGD met jour les paramtres du modle en fonction du gradient de la fonction de perte par rapport aux paramtres.

Formule :

(t+1) = (t) - L(t)

In[113]:
class SGD(Optimizer):
    """
    Stochastic Gradient Descent (SGD) optimizer.

    This class implements the SGD optimization algorithm, which updates
    parameters using their gradients and a fixed learning rate.

    Attributes:
        params (Union[List[object], object]): Parameters to optimize.
        learning_rate (float): The learning rate for the optimizer.
    """

    def __init__(self, params: Union[List[object], object], learning_rate: float = 0.01, **kwargs) -> None:
        """
        Initializes the SGD optimizer with parameters, a learning rate, and additional options.

        Args:
            params (Union[List[object], object]): Parameters to optimize. Can be a list or a single object.
            learning_rate (float): Learning rate for the optimization algorithm. Default is 0.01.
            **kwargs: Additional arguments (reserved for future extensions or compatibility).
        """
        super().__init__(params, learning_rate)

    def step(self) -> None:
        """
        Performs a single optimization step.

        Updates the parameters by subtracting the product of the learning rate
        and their respective gradients.

        Raises:
            AttributeError: If a parameter does not have `data` or `grad` attributes.
        """
        for param in self.params:
            param.data -= self.learning_rate * param.grad

4.2 RMSProp

RMSProp adapte le taux dapprentissage pour chaque paramtre en fonction de la moyenne des carrs des gradients passs.

Formule :

(t+1) = (t) - (/(Eg + )) L(t)

class RMSProp(Optimizer):
    """
    RMSProp optimizer.

    This class implements the RMSProp optimization algorithm, which adjusts
    the learning rate for each parameter individually using a moving average
    of squared gradients.

    Attributes:
        params (Union[List[object], object]): Parameters to optimize.
        learning_rate (float): The learning rate for the optimizer.
        decay (float): The decay rate for the moving average of squared gradients.
        epsilon (float): A small value to prevent division by zero.
        g (List[float]): Moving average of squared gradients for each parameter.
    """

    def __init__(
        self,
        params: Union[List[object], object],
        learning_rate: float = 0.01,
        decay: float = 0.9,
        epsilon: float = 1e-8,
        **kwargs
    ) -> None:
        """
        Initializes the RMSProp optimizer.

        Args:
            params (Union[List[object], object]): Parameters to optimize. Can be a list or a single object.
            learning_rate (float): Learning rate for the optimization algorithm. Default is 0.01.
            decay (float): Decay rate for the moving average of squared gradients. Default is 0.9.
            epsilon (float): A small value to prevent division by zero. Default is 1e-8.
            **kwargs: Additional arguments (reserved for future extensions or compatibility).
        """
        super().__init__(params, learning_rate)
        self.decay = decay
        self.epsilon = epsilon
        self.g = [0.0 for _ in (self.params if isinstance(self.params, list) else [self.params])]

    def step(self) -> None:
        """
        Performs a single optimization step.

        Updates the parameters by adjusting their learning rate based on the
        moving average of squared gradients.

        Raises:
            AttributeError: If a parameter does not have `data` or `grad` attributes.
        """
        for i, param in enumerate(self.params if isinstance(self.params, list) else [self.params]):
            self.g[i] = self.decay * self.g[i] + (1 - self.decay) * param.grad ** 2
            param.data -= (self.learning_rate / (np.sqrt(self.g[i]) + self.epsilon)) * param.grad

class Adagrad(Optimizer):
    """
    Adagrad optimizer.

    This class implements the Adagrad optimization algorithm, which adapts
    the learning rate for each parameter based on the cumulative sum of
    squared gradients.

    Attributes:
        params (Union[List[object], object]): Parameters to optimize.
        learning_rate (float): The learning rate for the optimizer.
        epsilon (float): A small value to prevent division by zero.
        gt (List[float]): Cumulative sum of squared gradients for each parameter.
    """

    def __init__(
        self,
        params: Union[List[object], object],
        learning_rate: float = 0.01,
        epsilon: float = 1e-8,
        **kwargs
    ) -> None:
        """
        Initializes the Adagrad optimizer.

        Args:
            params (Union[List[object], object]): Parameters to optimize. Can be a list or a single object.
            learning_rate (float): Learning rate for the optimization algorithm.
            epsilon (float): A small value to prevent division by zero. Default is 1e-8.
            **kwargs: Additional arguments (reserved for future extensions or compatibility).
        """
        super().__init__(params, learning_rate)
        self.epsilon = epsilon
        self.gt = [0.0 for _ in (self.params if isinstance(self.params, list) else [self.params])]

    def step(self) -> None:
        """
        Performs a single optimization step.

        Updates the parameters by adjusting their learning rate based on the
        cumulative sum of squared gradients.

        Raises:
            AttributeError: If a parameter does not have `data` or `grad` attributes.
        """
        for i, param in enumerate(self.params if isinstance(self.params, list) else [self.params]):
            grad = param.grad
            self.gt[i] += grad ** 2
            param.data -= self.learning_rate / np.sqrt(self.gt[i] + self.epsilon) * grad

class Adam(Optimizer):
    """
    Adam optimizer.

    This class implements the Adam optimization algorithm, which combines
    the advantages of RMSProp and momentum-based optimization.

    Attributes:
        params (Union[List[object], object]): Parameters to optimize.
        learning_rate (float): The learning rate for the optimizer.
        beta1 (float): Coefficient for computing running averages of gradients. Default is 0.9.
        beta2 (float): Coefficient for computing running averages of squared gradients. Default is 0.999.
        eps (float): A small value to prevent division by zero. Default is 1e-8.
        m (List[float]): Exponential moving averages of gradients for each parameter.
        v (List[float]): Exponential moving averages of squared gradients for each parameter.
    """

    def __init__(
        self,
        params: Union[List[object], object],
        learning_rate: float = 0.001,
        beta1: float = 0.9,
        beta2: float = 0.999,
        eps: float = 1e-8,
        **kwargs
    ) -> None:
        """
        Initializes the Adam optimizer.

        Args:
            params (Union[List[object], object]): Parameters to optimize. Can be a list or a single object.
            learning_rate (float): Learning rate for the optimization algorithm. Default is 0.001.
            beta1 (float): Coefficient for computing running averages of gradients. Default is 0.9.
            beta2 (float): Coefficient for computing running averages of squared gradients. Default is 0.999.
            eps (float): A small value to prevent division by zero. Default is 1e-8.
            **kwargs: Additional arguments (reserved for future extensions or compatibility).
        """
        super().__init__(params, learning_rate)
        self.beta1 = beta1
        self.beta2 = beta2
        self.eps = eps
        self.m = [0.0 for _ in (self.params if isinstance(self.params, list) else [self.params])]
        self.v = [0.0 for _ in (self.params if isinstance(self.params, list) else [self.params])]

    def step(self) -> None:
        """
        Performs a single optimization step.

        Updates the parameters using the Adam optimization algorithm, which
        combines first- and second-moment estimates of the gradients.

        Raises:
            AttributeError: If a parameter does not have `data` or `grad` attributes.
        """
        for i, param in enumerate(self.params if isinstance(self.params, list) else [self.params]):
            grad = param.grad
            self.m[i] = self.beta1 * self.m[i] + (1 - self.beta1) * grad
            self.v[i] = self.beta2 * self.v[i] + (1 - self.beta2) * (grad ** 2)
            param.data -= self.learning_rate / (np.sqrt(self.v[i]) + self.eps) * self.m[i]

class AdamW(Optimizer):
    """
    AdamW optimizer.

    This class implements the AdamW optimization algorithm, which is a variant
    of Adam that decouples weight decay from the gradient-based optimization.

    Attributes:
        params (Union[List[object], object]): Parameters to optimize.
        learning_rate (float): The learning rate for the optimizer.
        beta1 (float): Coefficient for computing running averages of gradients. Default is 0.9.
        beta2 (float): Coefficient for computing running averages of squared gradients. Default is 0.999.
        eps (float): A small value to prevent division by zero. Default is 1e-8.
        weight_decay (float): Weight decay coefficient for regularization. Default is 1e-2.
        m (List[float]): Exponential moving averages of gradients for each parameter.
        v (List[float]): Exponential moving averages of squared gradients for each parameter.
    """

    def __init__(
        self,
        params: Union[List[object], object],
        learning_rate: float = 0.001,
        beta1: float = 0.9,
        beta2: float = 0.999,
        eps: float = 1e-8,
        weight_decay: float = 1e-2,
        **kwargs
    ) -> None:
        """
        Initializes the AdamW optimizer.

        Args:
            params (Union[List[object], object]): Parameters to optimize. Can be a list or a single object.
            learning_rate (float): Learning rate for the optimization algorithm. Default is 0.001.
            beta1 (float): Coefficient for computing running averages of gradients. Default is 0.9.
            beta2 (float): Coefficient for computing running averages of squared gradients. Default is 0.999.
            eps (float): A small value to prevent division by zero. Default is 1e-8.
            weight_decay (float): Weight decay coefficient for regularization. Default is 1e-2.
            **kwargs: Additional arguments (reserved for future extensions or compatibility).
        """
        super().__init__(params, learning_rate)
        self.beta1 = beta1
        self.beta2 = beta2
        self.eps = eps
        self.weight_decay = weight_decay
        self.m = [0.0 for _ in (self.params if isinstance(self.params, list) else [self.params])]
        self.v = [0.0 for _ in (self.params if isinstance(self.params, list) else [self.params])]

    def step(self) -> None:
        """
        Performs a single optimization step.

        Updates the parameters using the AdamW optimization algorithm, which
        incorporates weight decay directly into the parameter updates.

        Raises:
            AttributeError: If a parameter does not have `data` or `grad` attributes.
        """
        for i, param in enumerate(self.params if isinstance(self.params, list) else [self.params]):
            grad = param.grad
            # Compute weight-decayed gradient
            grad += self.weight_decay * param.data
            # Update moving averages
            self.m[i] = self.beta1 * self.m[i] + (1 - self.beta1) * grad
            self.v[i] = self.beta2 * self.v[i] + (1 - self.beta2) * (grad ** 2)
            # Update parameter data
            param.data -= self.learning_rate / (np.sqrt(self.v[i]) + self.eps) * self.m[i]

def func_nn(
    x: Union[float, Tensor], 
    W1: Tensor, 
    b1: Tensor, 
    W2: Tensor, 
    b2: Tensor
) -> Tensor:
    """
    A simple feed-forward neural network with one hidden layer.

    Args:
        x (Union[float, Tensor]): Input value, either a scalar (float) or a Tensor.
        W1 (Tensor): Weights of the first layer.
        b1 (Tensor): Bias of the first layer.
        W2 (Tensor): Weights of the second layer.
        b2 (Tensor): Bias of the second layer.

    Returns:
        Tensor: The output of the network after applying linear transformation,
                ReLU activation, and a second linear transformation.
    """
    x = Tensor(x)  # Ensures input is a Tensor.
    h1 = W1 * x + b1  # First layer computation.
    h1 = h1.relu()  # Apply ReLU activation.
    y = W2 * h1 + b2  # Second layer computation.
    return y

def mse(y: Tensor, y_hat: Tensor) -> Tensor:
    """
    Mean Squared Error (MSE) loss function.

    Args:
        y (Tensor): True labels.
        y_hat (Tensor): Predicted labels.

    Returns:
        Tensor: The MSE loss computed as the squared difference between
                true and predicted values.
    """
    return (y - y_hat) ** 2  # MSE loss computation.

def eval_nn_optim(
    x: List[Union[float, Tensor]],
    y: List[Union[float, Tensor]],
    optimizer_kwargs: dict,
    nb_iter: int,
    optimizer_type: Type[Optimizer] = SGD
) -> Tuple[Tensor, Tensor, Tensor, Tensor, List[float]]:
    """
    Evaluates the performance of an optimizer on a simple neural network.

    Args:
        x (List[Union[float, Tensor]]): Input data.
        y (List[Union[float, Tensor]]): True labels (targets).
        optimizer_kwargs (dict): Arguments to initialize the optimizer.
        nb_iter (int): Number of optimization iterations.
        optimizer_type (Type[Optimizer]): The optimizer class to use. Default is SGD.

    Returns:
        Tuple[Tensor, Tensor, Tensor, Tensor, List[float]]: A tuple containing:
            - The learned weights and biases: W1, b1, W2, b2.
            - A list of the average loss values at each iteration.
    """
    # Initialize parameters with random normal values.
    W1, b1 = Tensor(np.random.normal()), Tensor(np.random.normal())
    W2, b2 = Tensor(np.random.normal()), Tensor(np.random.normal())
    
    # Initialize optimizer with parameters and optimizer arguments.
    optimizer = optimizer_type([W1, b1, W2, b2], **optimizer_kwargs)
    
    losses = []
    
    for _ in tqdm(range(nb_iter)):
        total_loss = 0
        for i in range(len(x)):
            optimizer.zeros_grad()  # Zero the gradients.
            y_pred = func_nn(x[i], W1, b1, W2, b2)  # Forward pass.
            loss = mse(y[i], y_pred)  # Compute loss.
            loss.backward()  # Backpropagate the gradients.
            total_loss += loss.data  # Accumulate loss.
            optimizer.step()  # Update parameters.
            
        avg_loss = total_loss / len(x)  # Calculate average loss.
        losses.append(avg_loss)

    return W1, b1, W2, b2, losses

class LRScheduler:
    """
    Learning Rate Scheduler.

    This class provides a mechanism to adjust the learning rate during training.

    Attributes:
        optimizer (Optimizer): The optimizer whose learning rate is being adjusted.
        learning_rate (float): The current learning rate.

    Methods:
        get_last_lr() -> float:
            Returns the last used learning rate.
        get_lr() -> float:
            Returns the current learning rate based on the scheduling rule.
        step() -> None:
            Updates the learning rate using `get_lr()` and applies it to the optimizer.
    """

    def __init__(self, optimizer: Optimizer, initial_lr: float) -> None:
        """
        Initializes the learning rate scheduler.

        Args:
            optimizer (Optimizer): The optimizer to adjust the learning rate for.
            initial_lr (float): The initial learning rate to use.

        self.learning_rate = initial_lr

        """
        Returns the last used learning rate.

        Returns:
            float: The last learning rate used by the scheduler.
        """

        """
        Computes and returns the current learning rate.

        This method should be overridden in a subclass to implement the

        Returns:
            float: The current learning rate.
        """
        raise NotImplementedError("This method should be overridden by subclasses.")

    def step(self) -> None:
        """
        Updates the learning rate based on `get_lr()` and applies it to the optimizer.

        This method should be called at each training step to adjust the learning rate.
        """

class LRSchedulerOnPlateau(LRScheduler):
    """
    Learning Rate Scheduler that reduces the learning rate when the validation loss plateaus.

    This class adjusts the learning rate based on the validation loss. If the validation loss
    does not improve for a specified number of iterations (patience), the learning rate is reduced
    by a specified factor.

    Attributes:
        patience (int): The number of iterations with no improvement before reducing the learning rate.
        factor (float): The factor by which the learning rate is multiplied when it is reduced.
        min_lr (float): The minimum learning rate that can be reached.
        mode (Literal['min', 'max']): Whether to monitor the validation loss for a minimum ('min') or maximum ('max').
        threshold (float): The threshold for considering an improvement in validation loss.
        min_loss (float or None): The best validation loss observed so far, adjusted by the threshold.
        bad_loss (int): The number of consecutive epochs without improvement in the validation loss.

    Methods:
        get_lr(val_loss: float) -> float:
            Computes the new learning rate based on the validation loss.
        step(val_loss: float) -> None:
            Updates the learning rate based on the validation loss and applies it to the optimizer.
    """

    def __init__(
        self, 
        optimizer: Optimizer, 
        initial_lr: float, 
        patience: int = 10, 
        factor: float = 0.1, 
        min_lr: float = 1e-6, 
        mode: Literal['min', 'max'] = 'min', 
        threshold: float = 1e-4
    ) -> None:
        """
        Initializes the LRSchedulerOnPlateau.

        Args:
            optimizer (Optimizer): The optimizer to adjust the learning rate for.
            initial_lr (float): The initial learning rate to use.
            patience (int, optional): The number of iterations with no improvement before reducing the learning rate. Default is 10.
            factor (float, optional): The factor by which to reduce the learning rate. Default is 0.1.
            min_lr (float, optional): The minimum learning rate. Default is 1e-6.
            mode (Literal['min', 'max'], optional): Whether to monitor for 'min' or 'max' validation loss. Default is 'min'.
            threshold (float, optional): The threshold for improvement in validation loss. Default is 1e-4.
        """
        super().__init__(optimizer, initial_lr)
        self.patience = patience
        self.factor = factor
        self.min_lr = min_lr
        self.mode = mode
        self.threshold = threshold
        self.min_loss = None
        self.bad_loss = 0

    def get_lr(self, val_loss: float) -> float:
        """
        Computes the new learning rate based on the validation loss.

        If the validation loss does not improve for a number of iterations defined by `patience`,
        the learning rate is reduced by the `factor`.

        Args:
            val_loss (float): The current validation loss.

        Returns:
            float: The updated learning rate.
        """
        lr = self.learning_rate
        if self.mode == 'max':
            val_loss = -val_loss  # In case we're maximizing, invert the loss.

        if self.min_loss is None:
            self.min_loss = val_loss - self.threshold
        elif val_loss < self.min_loss:
            self.bad_loss = 0
            self.min_loss = val_loss - self.threshold
        else:
            self.bad_loss += 1

        if self.bad_loss > self.patience and self.learning_rate > self.min_lr:
            self.bad_loss = 0
            lr = self.learning_rate * self.factor  # Reduce learning rate.

        return lr

    def step(self, val_loss: float) -> None:
        """
        Updates the learning rate based on the validation loss and applies it to the optimizer.

        Args:
            val_loss (float): The current validation loss used to adjust the learning rate.
        """
        self.learning_rate = self.get_lr(val_loss)
        self.optimizer.learning_rate = self.learning_rate

def clip_gradients(tensor, min_val=-1e3, max_val=1e3):
    """Clip gradient values to prevent overflow"""
    if isinstance(tensor.grad, np.ndarray):
        tensor.grad = np.clip(tensor.grad, min_val, max_val)
    else:
        tensor.grad = max(min(tensor.grad, max_val), min_val)
